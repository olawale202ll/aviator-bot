# ✈️ Aviator Signal Bot

A Telegram bot that generates Aviator-style signals based on simple logic and automation.

## 🚀 Features

- 📡 Auto signal generation
- 🎯 Manual signal command
- 🤖 Telegram bot integration
- ⚡ Fast deployment (Railway)

## 🛠 Commands

- /start → Start the bot
- /signal → Get a signal
- /autosignal → Start auto signals
